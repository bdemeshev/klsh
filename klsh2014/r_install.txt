Инструкция по установке R, R-studio и доп. пакетов.

1. R и R-studio --- бесплатные программы, скачить их можно свободно по адресам:
http://cran.rstudio.com/bin/windows/base/
http://www.rstudio.com/products/rstudio/download/
2. Устанавливать нужно сначала R, потом R-studio
3. ВАЖНО! Если имя пользователя Windows записано русскими буквами, 
то при установке R и R-studio лучше указать такие папки для установки:
c:\R\ --- для R
c:\Rstudio\ --- для R-studio
Если имя пользователя записано английскими буквами, то можно не заморачиваться.
4. Дополнительные пакеты ставятся после установки R-studio.
Например, чтобы поставить пакет erer нужно:
4.1. убедиться что комп подключен к интернету
4.2. запустить Rstudio
4.3. Выбрать Tools --- Install packages
4.4. Набрать "erer", нажать "Install"
Аналогично устанавливаются и другие пакеты.
